# node-reloader
auto reloader &amp; start script for node js

## Usage
- Merge with your bin folder.
- Configure your package.json like below:
<pre><code>
"scripts": {
      "start": "sh bin/service.sh start",
      "stop": "sh bin/service.sh stop",
      "restart": "sh bin/service.sh restart"
  }
</code></pre>
